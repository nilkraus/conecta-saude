Conecta Saúde

Sistema web desenvolvido com foco em **HTML5 semântico, acessibilidade e organização de dados**, simulando um portal de cadastro de pacientes e visualização de horários médicos.

---

 Sobre o Projeto

O **Conecta Saúde** é um projeto front-end que representa o módulo inicial de um sistema para clínicas médicas, permitindo o cadastro de pacientes e a visualização de horários de atendimento dos médicos.

O projeto foi desenvolvido com ênfase em:

- Estrutura semântica em HTML5  
- Acessibilidade para diferentes tipos de usuários  
- Organização clara de formulários e tabelas  
- Experiência do usuário (UX)  

---

Funcionalidades

Cadastro de pacientes com validação HTML5  
Organização de dados com `<fieldset>` e `<legend>`  
Tabela de horários médicos estruturada e acessível  
Navegação entre páginas (cadastro, horários e perfis médicos)  
Estrutura preparada para leitores de tela  

Acessibilidade

Este projeto segue boas práticas de acessibilidade:

- Uso correto de `<label>` associado aos inputs  
- Campos obrigatórios claramente definidos  
- Estrutura com `<header>`, `<main>`, `<section>`  
- Tabela com `<thead>`, `<tbody>` e `scope`  
- Navegação clara e consistente  

Tecnologias Utilizadas

- HTML5  
- CSS3
